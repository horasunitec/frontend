# Manejo Horas de Vinculacion UNITEC
Aplicacion web para manejar las horas de trabajo social de los alumnos de UNITEC.

## Getting Started

### Build and Run (Port 3000)

```
$ npm install
$ npm start
```

### Generators

```
$ npm install -g plop
$ plop
```

### Tests

```
$ npm install -g karma
$ npm test
```
