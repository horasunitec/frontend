Bienvenido Admin!

Descripcion de Modulos

Proyectos

Pantalla para la vista, creacion y edicion de proyectos, en esta pantalla se muestran los proyectos del periodo actual.
Los proyectos contienen nombre, descripcion de proyecto, carreras asociadas y Oraganizacion Beneficiaria


Secciones

Pantalla para la vista, creacion y edicion de secciones, en esta pantalla se muestran las Secciones del periodo actual.
Las secciones contiene codigo, clase, docente y alumnos


Periodos

Pantalla para la vista del periodo actual y de los demas periodos anteriores.
Cada vez que comienza un nuevo periodo se debe crear y asignar el periodo.


Clases

Pantalla para la vista, creacion y edicion de las clases.
Las clases contienen codigo de clase y Nombre.


Generar Finiquitos

Pantalla para la generacion del documento final de Finiquito para los alumnos con las horas completadas.


Profesores

Pantalla para la vista, creacion y edicion de Profesores.
Los profesores contienen numero de cuenta, nombre, correo y campus al que pertenece.
NOTA: El numero de cuenta del docente es el de Talento Humano(T.H.)


Alumnos

Pantalla para la vista, creacion y edicion de Alumnos.
Los alumnos contienen numero de cuenta, nombre, correo, carrera y campus al que pertenece.


Aprobar Horas

Pantalla para vista de los proyectos con horas pendientes de aprobar.


Reportes

Pantalla para la generacion de distintos reportes.
