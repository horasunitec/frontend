Bienvenido Admin!

Descripción de Modulos

Proyectos

Pantalla para la vista, creación y edición de proyectos. En esta pantalla se muestran los proyectos del período actual.
Los proyectos contienen nombre, descripción de proyecto, carreras asociadas y organización beneficiaria


Secciones

Pantalla para la vista, creación y edición de secciones. En esta pantalla se muestran las secciones del período actual.
Las secciones contiene código, clase, docente y alumnos


Períodos

Pantalla para la vista del período actual y de los demás períodos anteriores.
Cada vez que comienza un nuevo período se debe crear y asignar el período.


Clases

Pantalla para la vista, creación y edición de las clases.
Las clases contienen código de clase y nombre.


Generar Finiquitos

Pantalla para la generación del documento final de Finiquito para los alumnos con las horas completadas.


Finiquitados

Pantalla para la generación del documento final de Finiquito para los alumnos con las horas completadas que ya se han generado anteriormente.


Profesores

Pantalla para la vista, creación y edición de Profesores.
Los profesores contienen número de cuenta, nombre, correo y campus al que pertenece.
NOTA: El número de cuenta del docente es el de Talento Humano(T.H.)


Alumnos

Pantalla para la vista, creación y edición de Alumnos.
Los alumnos contienen número de cuenta, nombre, correo, carrera y campus al que pertenece.


Aprobar Horas

Pantalla para vista de los proyectos con horas pendientes de aprobar.


Reportes

Pantalla para la generación de distintos reportes.
